# Technologies
- C++ standard 17
- cmake
- make

# Setup
```sh
$ cd 
$ sudo apt install g++
$ sudo apt install make
$ sudo apt install cmake
```

# Build
```sh
mkdir build
cd build
cmake ..
cmake --build .
```

> For running this code you just have to write ./segmenttree
